# ionic-myTabApp
Sample ionic app my tab

### Commands:
- Tabs template
  ```bash
  ionic start myApp tabs
  ```
- Serve app
  ```bash
  ionic serve
  ```
- Build for Android
  ```
  ionic build android
  ```
